# CMS-9115

CMS-9115-F — Medicare and Medicaid Programs; Patient Protection and Affordable Care Act; Interoperability and Patient Access Final Rule. Published in the Federal Register on May 1, 2020 ([85 FR 25510](https://www.federalregister.gov/documents/2020/05/01/2020-05050)).

Two APIs that Payerbox supports trace to this rule: the Patient Access API and the Provider Directory API. Both remain active under CMS-0057-F.

## Regulated entities

| Entity type | Applies | Citation |
|---|---|---|
| Medicare Advantage organizations | Yes | 42 CFR 422.119, 422.120 |
| State Medicaid FFS programs | Yes | 42 CFR 431.60, 431.70 |
| Medicaid managed care plans | Yes | 42 CFR 438.62(b)(1)(ii), 438.242(b)(6) |
| CHIP FFS programs | Yes | 42 CFR 457.730, 457.760 |
| CHIP managed care plans | Yes | 42 CFR 457.1233(d) |
| QHP issuers on the Federally Facilitated Exchanges | Patient Access only (not Provider Directory) | 45 CFR 156.221 |

Out of scope: employer-sponsored commercial plans, state-based exchange QHPs, stand-alone dental plans.

## Patient Access API

Members access claims, encounters with provider remittances and enrollee cost-sharing, and USCDI clinical data through a member-authorized third-party app.

- **Effective:** January 1, 2021. CMS enforcement began July 1, 2021.
- **Claims scope:** service dates from January 1, 2016 onward.
- **Clinical scope:** USCDI per 45 CFR 170.213. USCDI v3 takes effect January 1, 2026 under ONC HTI-1.
- **MA-PD addition:** formulary data.
- **CMS-0057-F extension:** prior authorization information added to Patient Access scope, effective January 1, 2027 — see [CMS-0057](cms-0057.md).

See [Interop APIs / Patient Access](../interop-apis/patient-access.md).

## Provider Directory API

Any caller queries the payer's network directory. No authentication required.

- **Effective:** January 1, 2021.
- **Update cadence:** changes within 30 calendar days of receipt.
- **Minimum data set:** name, address, phone, specialty. Pharmacy data for MA-PD.
- **Note:** QHP issuers on FFEs were excluded from this requirement in 9115 and remain excluded under 0057.

See [Interop APIs / Provider Directory](../interop-apis/provider-directory.md).
