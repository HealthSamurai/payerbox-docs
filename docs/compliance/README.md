# Compliance

Regulatory anchors, reporting requirements, and attestation guidance for CMS-0057-F and CMS-9115-F.

| Page | What it covers |
|---|---|
| [CMS-9115](cms-9115.md) | 2020 Interoperability and Patient Access Final Rule |
| [CMS-0057](cms-0057.md) | 2024 Interoperability and Prior Authorization Final Rule |
| [Reporting](reporting.md) | Patient Access annual metrics and public prior authorization metrics |

Both rules apply concurrently. CMS-0057-F amends and extends CMS-9115-F; it does not replace it. APIs first established by CMS-9115-F (Patient Access, Provider Directory) remain in force.
