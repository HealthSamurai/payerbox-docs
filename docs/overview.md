# Overview

Payerbox is a FHIR compliance platform for health plans, covering CMS-0057-F and CMS-9115-F requirements. It is composed of four Docker images: Aidbox (FHIR engine), Interop (Patient Access, Provider Access, Payer-to-Payer, Provider Directory APIs), Prior Auth (CRD, DTR, PAS), and Portals (Admin Portal, Developer Portal, Smart App Gallery).

This documentation is under active development.
